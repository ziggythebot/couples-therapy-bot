# Contributing

This is an early-stage research project exploring AI-powered couples therapy. Contributions, feedback, and ideas are welcome.

## How to Contribute

### Ideas & Feedback
- Open an issue to discuss therapeutic models, interview structures, or technical approaches
- Share research on couples therapy frameworks (Gottman, EFT, attachment theory, etc.)
- Suggest improvements to session design or chat analysis

### Code Contributions
Once implementation begins:
1. Fork the repo
2. Create a feature branch (`git checkout -b feature/your-idea`)
3. Commit changes (`git commit -m 'Add feature'`)
4. Push to branch (`git push origin feature/your-idea`)
5. Open a Pull Request

### Documentation
- Expand therapeutic model documentation
- Add session templates or question flows
- Improve architecture diagrams

## Ethical Considerations

This bot is designed to **supplement, not replace** professional therapy:
- Not a substitute for crisis intervention or severe relationship issues
- Users should be informed of system limitations
- Privacy and data security are critical (sensitive relationship data)
- Therapeutic accuracy matters—any model implementation should be evidence-based

## Questions?

Open an issue or reach out via [GitHub Discussions](https://github.com/ziggythebot/couples-therapy-bot/discussions).
